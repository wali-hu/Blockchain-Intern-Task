# Blockchain Developer Intern Tasks

This repository contains all blockchain development tasks completed during my internship at ProgrammX. Each branch represents a different project focusing on various blockchain technologies including Solana, Ethereum, and cross-chain protocols.

##  Projects Overview

### Solana Development
- **[solana-wallet](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/solana-wallet)** - Command-line wallet for Solana devnet with balance checking and transaction capabilities
- **[solana-spl-ts](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/solana-spl-ts)** - TypeScript CLI for SPL token operations (create, mint, transfer)
- **[solana-token-2022](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/solana-token-2022)** - Implementation using Solana's Token-2022 program with advanced features
- **[send-sol-versioned-transactios](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/send-sol-versioned-transactios)** - Versioned (v0) transaction implementation for SOL transfers on devnet
- **[jupiter-swap](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/jupiter-swap)** - Integration with Jupiter aggregator for Solana token swaps
- **[transaction_simulation](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/transaction_simulation)** - SPL token CLI with transaction simulation capabilities

### Solana Anchor Programs
- **[task/CURD-Solana-Anchor](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/CURD-Solana-Anchor)** - Full-stack CRUD application with Anchor program and REST API
- **[task/anchor-card-minting-program](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/anchor-card-minting-program)** - On-chain card store that mints SPL tokens (Gold, Platinum, Silver) for SOL payments
- **[task/Deserializing-Account-Data-on-Solana](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/Deserializing-Account-Data-on-Solana)** - Account data deserialization examples and utilities

### Ethereum & EVM Development
- **[task/0x-swap-feature](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/0x-swap-feature)** - Terminal client and REST API for 0x Swap with Permit2 support
- **[task/1155-multimint-amoy](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/1155-multimint-amoy)** - Gas-efficient ERC-1155 batch minting contract using Foundry
- **[task/foundry-DAO](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/foundry-DAO)** - On-chain governance system using OpenZeppelin's Governor and ERC20Votes

### Cross-Chain & DeFi Integration
- **[task/rubic-swap-simulator](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/rubic-swap-simulator)** - Rubic.finance swap simulation tool for pre-trade analysis
- **[task/four-meme-buy-sell-simulator](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/four-meme-buy-sell-simulator)** - BSC interaction scripts for Four.meme platform

### Backend & Database
- **[task/mongo-token-crud](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/task/mongo-token-crud)** - MongoDB integration for token data management

### Learning & Fundamentals
- **[Rust_Learnings](https://github.com/ptechfusion19/Blockchain-Intern-Task/tree/Rust_Learnings)** - Rust programming fundamentals (variables, functions, data types)
